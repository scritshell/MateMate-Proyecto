MateMate - Tu Compañero de Estudio de Ajedrez
Una aplicación móvil moderna para el estudio y práctica de ajedrez, desarrollada con Kotlin y Android Studio.

Características Implementadas:
Navegación Principal
Navigation Drawer con 5 secciones principales

Fragments: Inicio, Aperturas, Tácticas, Bloc de Notas, Comunidad

Navegación fluida entre todas las pantallas

Módulos de Estudio:
Aperturas: Catálogo completo de aperturas de ajedrez

Tácticas: Ejercicios de mate en 1, 2, 3

Bloc de Notas: Gestión de análisis y partidas personales

Práctica Libre: Tablero interactivo para estudio

Interfaz de Usuario:
Material Design moderno y responsive

CardViews y RecyclerViews optimizados

Navegación intuitiva y fluida

Tecnologías Utilizadas
Kotlin - Lenguaje principal

Android Studio - IDE de desarrollo

Navigation Component - Gestión de navegación

View Binding - Binding seguro de vistas

RecyclerView - Listas eficientes

Material Design - Componentes UI modernos

CardView - Tarjetas visuales

Funcionalidades por Módulo:
- InicioFragment
Dashboard con estadísticas rápidas

Accesos directos a todas las funcionalidades

Seguimiento de progreso

- AperturasFragment
Lista de aperturas organizadas

Información detallada de cada apertura

Navegación al tablero de práctica

- TacticasFragment
Ejercicios organizados por dificultad

Niveles: Mate en 1, Mate en 2, Mate en 3

Seguimiento de estadísticas

- BlocNotasFragment
Gestión de notas de estudio

Análisis de partidas personales

Organización por fecha y categoría

- SocialFragment
Interfaz para comunidad (en desarrollo)

Tabs para Feed, Amigos y Mensajes

Funcionalidades pendientes:

Análisis de partidas en tiempo real

Sistema social completo

Exportación de partidas en formato PGN
